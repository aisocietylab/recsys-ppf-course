# Website Page Starter

Click "Use this template" to create a new repository with the contents of this template, or click [here](https://github.com/new?owner=aisocietylab&template_name=website-page-starter-template&template_owner=aisocietylab).

To run execute `bundle exec jekyll serve` in the terminal and open `http://localhost:4000/` in your browser to view the website. You can edit the content of the website by modifying the files in this repository. The main content of the project page is located in `index.md`.
